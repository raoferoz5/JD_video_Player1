## 🎬 DJ Media Player: Immersive Video Experience
A sophisticated cross-platform video player designed for fluid, modern navigation.

### 🎯 The Project Goal
The DJ Media Player bridges the gap between traditional media players and modern social media interfaces. It provides users with a powerful playback engine while implementing an intuitive, vertical-swipe discovery mechanic.

### ✨ Key Features
TikTok-Style Navigation: Smooth vertical swiping logic for effortless video discovery.

MX Player Logic: Advanced gesture controls and playback features for a premium viewing experience.

Dynamic UI: Built using KivyMD to ensure a consistent, modern look across Android and Desktop.

Optimized Performance: Efficient handling of video assets and thumbnails for low-latency playback.
